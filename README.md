VaultMT

Современное экономическое ядро и API для серверов Minecraft, разработанное под платформы Paper / Folia (26.1.x). Проект спроектирован с упором на высокую производительность, модульность и использование современных стандартов Java 25.
✨ Особенности

    Полная поддержка Folia: Оптимизировано для многопоточной среды с флагом folia-supported: true.

    Гибридная архитектура (Bridge): Автоматическое обнаружение сторонних плагинов (например, EssentialsX) с возможностью переключения на встроенную автономную систему.

    Современный стек: Использование record для структуры транзакций и строгая типизация API.

    Безопасность: Все операции возвращают TransactionResult со строгим контролем статусов (успех, нехватка средств, отсутствие счета).

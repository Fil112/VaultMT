VaultMT — это решение нового поколения для серверов, которым нужна надежная, быстрая и расширяемая экономика. Мы ушли от монолитных устаревших систем, предложив модульный подход: плагин сам определяет лучший способ хранения данных, а администраторы получают полный контроль через удобный settings.yml.

Почему VaultMT?

    Высокая производительность: Оптимизировано для работы на актуальных версиях Java. Никаких «тяжелых» процессов, которые грузят сервер.

    Архитектура Bridge: Уникальная система провайдеров позволяет VaultMT работать как автономно, так и в связке с другими системами, если это необходимо.

    Гибкий конфиг: Настраивай всё: от символа валюты до интервалов автосохранения.

Основные особенности

    ✅ Автоматизация: Система сама подберет оптимальный провайдер для экономики.

    ✅ Легкость: Минимальный вес плагина, практически нулевое влияние на TPS сервера.

    ✅ Разработчикам: Чистое API для интеграции с твоими другими плагинами и ботами.

    ✅ Полная поддержка Folia: Оптимизировано для многопоточной среды.

    ✅ Гибридная архитектура (Bridge): Автоматическое обнаружение сторонних плагинов (например, EssentialsX) с возможностью переключения на встроенную автономную систему.
